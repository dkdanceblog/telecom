# Телекомыч ловит задачи

Финальная тестовая сборка.

- 1 день = 30 секунд.
- Вся рабочая неделя ПН–ПТ = 2 минуты 30 секунд.
- На старте есть инструкция.
- После прохождения пятницы появляется финальная заставка.
- По любой кнопке после финала — возврат на стартовый экран.

Управление:
- Любая кнопка запускает игру.
- ← / → или A / D — движение.
- ↑ — прыжок.
- ⏸ — пауза, продолжение через большую кнопку ▶.
- 🔊 / 🔇 — звук.

Механики:
- 📅 DAY OFF очищает экран и снижает стресс.
- ☎️ Созвон повышает стресс и замедляет игрока.
- 🔥 ASAP быстрее обычных задач, даёт больше очков и стресс.
- 📱 Сообщения лучше избегать.
- 😂 Мемы, еда, кофе и лайк от клиента помогают от выгорания.
- В пятницу задачи горят, но поток немного ослаблен по сравнению с тестовой мясорубкой.
- Пасхалка: раз в 50 секунд сбоку вылезает один из двух персонажей.
