# Телекомыч ловит задачи

Версия с рабочей неделей:
- 5 минут = 5 рабочих дней: ПН, ВТ, СР, ЧТ, ПТ.
- Поток задач постепенно растёт.
- Во вторник появляются сообщения.
- В среду появляются созвоны.
- В четверг появляются ASAP-задачи.
- В пятницу становится заметно сложнее.

Управление:
- Любая кнопка запускает игру.
- ← / → или A / D — движение.
- ↑ — прыжок.
- P / Esc — пауза.
- На телефоне: кнопки слева/справа, старт/прыжок, пауза.

Механики:
- 📅 DAY OFF убирает все объекты с экрана и снижает стресс.
- ☎️ Созвон повышает стресс и временно замедляет игрока.
- 🔥 ASAP падает очень быстро, даёт много очков, но повышает стресс.
- 📱 Сообщения лучше не ловить: они повышают стресс.
- 😂 Мемы снижают стресс.
- 🥐 Круассан сильнее снижает стресс.
- ☕ Кофе ускоряет игрока в 2 раза на время.
